# flask-heroku-api

Video Tutorial

https://youtu.be/dBXAOswc66o

Set it up!
 
`virtualenv venv && . venv/bin/activate`

`pip3 install -r requirements.txt`


Run it!

`python3 app.py`


Deploy it!


`heroku create APP_NAME`

`git push heroku main`


See your deployment!

https://APP_NAME.herokuapp.com/
